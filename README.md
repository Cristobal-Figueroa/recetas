# RecipeHub - Simple Recipe Finder

A clean, modern web application for finding and saving recipes. Built with React and Vite.

## Features

- 🔍 **Search Recipes** - Search by name or ingredient
- ⭐ **Save Favorites** - Save your favorite recipes to localStorage
- 🎲 **Random Recipe** - Get surprised with a random recipe
- 📱 **Responsive Design** - Works great on mobile, tablet, and desktop
- 🎨 **Modern UI** - Clean interface with smooth animations

## Technologies

- React 18
- Vite
- TheMealDB API
- CSS3 with CSS Variables
- LocalStorage for favorites

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## API

This project uses [TheMealDB API](https://www.themealdb.com/) - a free recipe database.
